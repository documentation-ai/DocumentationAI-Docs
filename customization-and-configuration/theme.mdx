---

```json
{
  "configType": "theme",
  "scope": "site-wide", 
  "configFile": "documentation.json",
  "properties": ["colors.light", "colors.dark"],
  "requirements": ["hex color format"]
}
```

Configure your documentation site's theme using the `colors` object in `documentation.json`. The theme system supports both light and dark modes with automatic contrast optimization.

## Light and Dark Mode Configuration

Set up dual themes that automatically adapt based on user preference or system settings.

```json
{
  "colors": {
    "light": {
      "brand": "#3143e3",
      "heading": "#1a1a1a", 
      "text": "#374151"
    },
    "dark": {
      "brand": "#85a1ff",
      "heading": "#f2f2f2",
      "text": "#c1c1c1"
    }
  }
}
```

## Theme Requirements

- Use hex color format for all color values (e.g., `#3143e3`)
- Colors automatically optimize for contrast and accessibility
- Both light and dark themes are optional but recommended for better user experience
- Brand colors should maintain adequate contrast ratios in their respective themes

<Callout>
The theme system automatically handles contrast optimization and ensures your colors maintain accessibility standards across both light and dark modes.
</Callout>
