This section will enumerate endpoints with methods, paths, auth, parameters, and responses.

Example structure:

```
GET /api/v1/documentations
Auth: Bearer <token>

Query
- page: integer
- per_page: integer

Responses
- 200: { items: [...], page: 1, per_page: 20 }
- 401: Unauthorized
```
