# On-Page AI Menu

Every documentation page published has the below three options if your users wants to use other AI tools 



every documentation page is available in md/markdownformat, probably the most optimised format for LLMs to understand the text. 

there are three page options availabke \
\
copy page as markdown - it copies the page page markdown content to the clipboard you can copy pase \
Open in Claude - directly open the claude with the .md url \
Open in Chatgpt - directly open the chatgpt with the .md url
