---
title: Create Your First Project
description: A beginner tutorial to create, write, and publish your first docs with Documentation.AI
---

### Goal

By the end, you will have a public documentation site with an introduction page and a components section.

### Prerequisites

- Account on Documentation.AI
- Git installed

### Steps

1. Create a project in the dashboard
2. Clone your `DocumentationAI-Docs` repo
3. Edit `introduction.mdx`
4. Commit and push
5. Publish from the dashboard

Next: configure a custom domain or invite your team.
