## Overview

Embed videos from platforms like YouTube, Vimeo, and Loom, or use self-hosted video files. The Video component handles both iframe embedding and HTML5 video playback.

Common use cases:
- Platform videos (YouTube, Vimeo, Loom)
- Self-hosted video files (MP4, WebM)
- Interactive content and external applications

<Video 
  src="https://www.youtube.com/embed/Reu01KxMSF0?si=Rf3PTfKp1hMn9qhX" 
  width="560" 
  height="315"
/>

## Video (iframe mode)

Use for YouTube, Vimeo, Loom, and other video platforms. This is the default mode for platform videos.

````jsx
<Video 
  src="https://www.youtube.com/embed/Reu01KxMSF0?si=Rf3PTfKp1hMn9qhX" 
  title="YouTube video player"
  width="560" 
  height="315"
/>
````

### Attributes

<ParamField path="src" param-type="string" required="true" showLocation="false">
  Video embed URL from platform.
</ParamField>

<ParamField path="width" param-type="string" required="false" showLocation="false">
  Video player width in pixels.
</ParamField>

<ParamField path="height" param-type="string" required="false" showLocation="false">
  Video player height in pixels.
</ParamField>

<ParamField path="title" param-type="string" required="false" showLocation="false">
  Accessible title for screen readers.
</ParamField>

<ParamField path="allow-full-screen" param-type="boolean" required="false" showLocation="false">
  Allow content to go fullscreen (default: `true`).
</ParamField>

<ParamField path="priority" param-type="boolean" required="false" showLocation="false">
  Load immediately instead of lazy loading (default: `false`).
</ParamField>

<ParamField path="style" param-type="string" required="false" showLocation="false">
  CSS styles for alignment and positioning.
</ParamField>

## Video (HTML5 mode)

Use for self-hosted video files (MP4, WebM, OGG) with native browser controls.

````jsx
<Video 
  src="https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4"
  width="560" 
  height="315"
  controls="true"
  poster="https://example.com/thumbnail.jpg"
/>
````

### Attributes

<ParamField path="src" param-type="string" required="true" showLocation="false">
  Direct URL to video file (MP4, WebM, OGG).
</ParamField>

<ParamField path="width" param-type="string" required="false" showLocation="false">
  Video player width in pixels.
</ParamField>

<ParamField path="height" param-type="string" required="false" showLocation="false">
  Video player height in pixels.
</ParamField>

<ParamField path="controls" param-type="boolean" required="false" showLocation="false">
  Show video controls (default: `true`).
</ParamField>

<ParamField path="autoplay" param-type="boolean" required="false" showLocation="false">
  Auto-play video when page loads (default: `false`).
</ParamField>

<ParamField path="muted" param-type="boolean" required="false" showLocation="false">
  Start video muted (default: `false`).
</ParamField>

<ParamField path="loop" param-type="boolean" required="false" showLocation="false">
  Loop video continuously (default: `false`).
</ParamField>

<ParamField path="poster" param-type="string" required="false" showLocation="false">
  Image URL to display before video starts playing.
</ParamField>

<ParamField path="preload" param-type="string" required="false" showLocation="false">
  How much video to preload: `none`, `metadata`, `auto`.
</ParamField>

<ParamField path="fetchPriority" param-type="string" required="false" showLocation="false">
  Browser loading priority: `high`, `low`, `auto`.
</ParamField>

<ParamField path="style" param-type="string" required="false" showLocation="false">
  CSS styles for alignment and positioning.
</ParamField>

## Iframe

Use for embedding external content like forms, tools, or interactive applications.

````jsx
<Iframe 
  src="https://docs.google.com/forms/d/e/1FAIpQLSexample/viewform" 
  title="Feedback form"
  width="100%" 
  height="600"
/>
````

### Attributes

<ParamField path="src" param-type="string" required="true" showLocation="false">
  URL of external content to embed.
</ParamField>

<ParamField path="width" param-type="string" required="false" showLocation="false">
  Iframe width (pixels or percentage).
</ParamField>

<ParamField path="height" param-type="string" required="false" showLocation="false">
  Iframe height (pixels or percentage).
</ParamField>

<ParamField path="title" param-type="string" required="false" showLocation="false">
  Accessible title for screen readers.
</ParamField>

<ParamField path="allow-full-screen" param-type="boolean" required="false" showLocation="false">
  Allow content to go fullscreen (default: `true`).
</ParamField>

<ParamField path="sandbox" param-type="string" required="false" showLocation="false">
  Security restrictions (default: `"allow-scripts allow-same-origin allow-presentation"`).
</ParamField>

<ParamField path="priority" param-type="boolean" required="false" showLocation="false">
  Load immediately instead of lazy loading (default: `false`).
</ParamField>

<ParamField path="style" param-type="string" required="false" showLocation="false">
  CSS styles for alignment and positioning.
</ParamField>

## Platform support

**Video platforms:** YouTube, Vimeo, Loom, Wistia  
**Video formats:** MP4, WebM, OGG  
**Common patterns:** Accessibility titles, dimension specifications for layout stability, priority loading for above-fold content
