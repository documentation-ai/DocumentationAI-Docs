## Images

Images enhance documentation by providing visual context and examples. Documentation.AI supports two methods for adding images: MDX Image component syntax and the visual editor.

### MDX Image component

Write the `<Image>` component directly in your MDX files to display images. The component accepts `src` and `alt` attributes.

```markdown
<Image 
  src="https://your-cdn.com/image.png" 
  alt="Description of the image" 
/>
```

### Image attributes and alignment

The `<Image>` component supports additional attributes for sizing and positioning.

#### Basic attributes

```markdown
<Image 
  src="https://your-cdn.com/image.png" 
  alt="Description" 
  width="400" 
  height="300"
/>
```

#### Left alignment (default)

```markdown
<Image 
  src="https://your-cdn.com/image.png" 
  alt="Left aligned image" 
  width="433" 
  height="247" 
  style="width: 433px; height: auto; cursor: pointer; margin: 0 auto 0 0;" 
/>
```

### Adding images through the editor

The recommended method is using the Documentation.AI editor for optimized images served from our CDN.

#### Method 1: Type command

In the editor, type `/image` to open the image selector.

<Image uid="4fd1b667-7e9f-4dd4-8ff2-978ef3c40750" src="https://fdjsuzcsbmpqsraaooua.supabase.co/storage/v1/object/public/documentation-images/org-53a37986-2c9e-4094-b9e8-1e1ffae9e9ee/doc-b389b141-ae58-4fd5-91f9-6702fae9ac58/1758639375212-c39us34hlx5-uploadimage.png" width="716" height="240" alt="" style="width: 716px; height: auto; cursor: pointer;" />

#### Method 2: Plus icon

Click the "+" icon in the editor toolbar and select the image element.

#### Method 3: Copy and paste

Copy an image from anywhere (screenshot, file, web) and paste it directly into the editor. The image will automatically upload to our CDN.

### Image requirements

- **URL format**: Only absolute URLs are supported

- **Alt text**: Required for all images (SEO and accessibility)

- **Formats**: JPEG, JPG, PNG, GIF, WebP, SVG, ICO

- **CDN**: Editor-uploaded images are automatically optimized

<Callout kind="warning" collapsed="false">
  External CDN images may not be optimized for performance. Use the editor upload for best results.
</Callout>
