# CodeBlock Component

The CodeBlock component displays syntax-highlighted code snippets with support for multiple programming languages, line numbers, and interactive features.

## Overview

Use CodeBlock components to showcase code examples, configuration files, and command-line instructions with proper syntax highlighting and formatting.

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `language` | `string` | Yes | Programming language for syntax highlighting |
| `children` | `string` | Yes | The code content to display |
| `filename` | `string` | No | Optional filename to display above the code |
| `showLineNumbers` | `boolean` | No | Whether to show line numbers |
| `highlightLines` | `number[]` | No | Array of line numbers to highlight |
| `className` | `string` | No | Additional CSS classes for styling |

## Usage Examples

### Basic Code Block

```mdx
<CodeBlock language="javascript">
{`function greet(name) {
  return \`Hello, \${name}!\`;
}`}
</CodeBlock>
```

### With Filename

```mdx
<CodeBlock language="typescript" filename="utils.ts">
{`export function formatDate(date: Date): string {
  return date.toISOString().split('T')[0];
}`}
</CodeBlock>
```

### With Line Numbers

```mdx
<CodeBlock language="python" showLineNumbers={true}>
{`def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)`}
</CodeBlock>
```

### With Highlighted Lines

```mdx
<CodeBlock language="bash" highlightLines={[2, 4]}>
{`npm install
npm run build
npm test
npm run deploy`}
</CodeBlock>
```

- Uses semantic code elements with proper ARIA labels
- Supports keyboard navigation and focus management
- Provides alternative text for syntax highlighting
- Compatible with screen readers

## Best Practices

- Choose the correct language for accurate syntax highlighting
- Keep code examples concise and focused
- Use meaningful filenames when provided
- Highlight important lines sparingly
- Ensure code examples are functional and tested
