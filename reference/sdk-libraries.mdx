# SDK Libraries

Documentation.AI provides official SDKs and libraries to integrate our platform into your applications, CI/CD pipelines, and development workflows. Build custom tools, automate documentation processes, and create seamless integrations.

## Available SDKs

### JavaScript/TypeScript SDK

Full-featured SDK for Node.js and browser environments:

**Installation:**
```bash
npm install @documentation-ai/sdk
# or
yarn add @documentation-ai/sdk
```

**Basic Usage:**
```typescript
import { DocumentationAI } from '@documentation-ai/sdk';

const client = new DocumentationAI({
  apiKey: process.env.DOCAI_API_KEY,
  projectId: 'your-project-id'
});

// Create a new page
const page = await client.pages.create({
  title: 'Getting Started',
  content: '# Welcome to our API\n\nThis guide will help you...',
  path: 'getting-started'
});

// Generate content with AI
const generatedContent = await client.ai.generate({
  prompt: 'Create API documentation for user authentication',
  type: 'api-reference',
  format: 'markdown'
});
```

### Python SDK

Comprehensive Python library for automation and data processing:

**Installation:**
```bash
pip install documentation-ai
```

**Basic Usage:**
```python
from documentation_ai import DocumentationAI

client = DocumentationAI(
    api_key=os.getenv('DOCAI_API_KEY'),
    project_id='your-project-id'
)

# Bulk upload documentation
pages = [
    {'title': 'Introduction', 'content': intro_content, 'path': 'intro'},
    {'title': 'API Reference', 'content': api_content, 'path': 'api'}
]

result = client.pages.bulk_create(pages)

# Generate documentation from code
docs = client.ai.generate_from_code(
    code_files=['./src/api.py', './src/models.py'],
    output_format='markdown',
    include_examples=True
)
```

### Go SDK

Lightweight Go library for backend integrations:

**Installation:**
```bash
go get github.com/documentation-ai/go-sdk
```

**Basic Usage:**
```go
package main

import (
    "context"
    "github.com/documentation-ai/go-sdk"
)

func main() {
    client := docai.NewClient("your-api-key", "your-project-id")
    
    // Create documentation page
    page := &docai.Page{
        Title:   "API Overview",
        Content: "# API Documentation\n\nWelcome to our API...",
        Path:    "api-overview",
    }
    
    result, err := client.Pages.Create(context.Background(), page)
    if err != nil {
        log.Fatal(err)
    }
    
    fmt.Printf("Created page: %s\n", result.ID)
}
```

### REST API

Direct HTTP API access for any programming language:


### Workflow Integration

Integrate with your development workflow:

**Git Integration:**
```bash

docai sync --repo https://github.com/user/docs-repo \
           --branch main \
           --path ./docs \
```

**CI/CD Pipeline Integration:**
```yaml

name: Update Documentation
on:
  push:
    paths: ['src/**', 'api/**']
jobs:
  update-docs:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Generate API Docs
        uses: documentation-ai/generate-action@v1
        with:
          api-key: ${{ secrets.DOCAI_API_KEY }}
          project-id: ${{ secrets.DOCAI_PROJECT_ID }}
          source-path: './src'
          output-path: './docs/api'
```

## Advanced Usage

### Custom Integrations

Build sophisticated integrations with advanced features:

**Webhook Handling:**
```typescript
import express from 'express';

const app = express();

// Handle Documentation.AI webhooks
app.post('/webhook/docai', (req, res) => {
  const event = req.body;
  
  switch (event.type) {
    case 'page.published':
      // Trigger deployment or notification
      deployDocumentation(event.data.page);
      break;
    case 'content.updated':
      // Sync with external systems
      syncWithCMS(event.data);
      break;
  }
  
  res.status(200).send('OK');
});
```

### Batch Processing

Handle large-scale documentation operations:

**Bulk Content Processing:**
```python
import asyncio
from documentation_ai import AsyncDocumentationAI

async def process_documentation():
    client = AsyncDocumentationAI(api_key=API_KEY)
    
    # Process multiple files concurrently
    tasks = []
    for file_path in documentation_files:
        task = client.ai.generate_from_file(file_path)
        tasks.append(task)
    
    results = await asyncio.gather(*tasks)
    
    # Bulk upload results
    await client.pages.bulk_create(results)

asyncio.run(process_documentation())
```

### Error Handling

Robust error handling and retry logic:

```python
from documentation_ai.exceptions import (
    APIError, 
    RateLimitError, 
    ValidationError
)

try:
    result = client.pages.create(page_data)
except RateLimitError as e:
    # Handle rate limiting
    time.sleep(e.retry_after)
    result = client.pages.create(page_data)
except ValidationError as e:
    # Handle validation errors
    print(f"Validation failed: {e.errors}")
except APIError as e:
    # Handle general API errors
    print(f"API error: {e.message}")
```

## Examples and Use Cases

### Documentation Automation

Automate documentation generation from code:

```typescript
// Generate API documentation from OpenAPI spec
const apiDocs = await client.ai.generateFromOpenAPI({
  specUrl: 'https://api.example.com/openapi.json',
  includeExamples: true,
  generateSDKExamples: ['javascript', 'python', 'curl']
});

// Create pages for each endpoint
for (const endpoint of apiDocs.endpoints) {
  await client.pages.create({
    title: `${endpoint.method.toUpperCase()} ${endpoint.path}`,
    content: endpoint.documentation,
    path: `api/${endpoint.operationId}`,
    tags: ['api', endpoint.method]
  });
}
```

### Content Synchronization

Keep documentation in sync with external sources:

```python
# Sync with database schema
def sync_database_docs():
    schema = get_database_schema()
    
    for table in schema.tables:
        content = client.ai.generate_table_docs(
            table_name=table.name,
            columns=table.columns,
            relationships=table.relationships
        )
        
        client.pages.upsert(
            path=f'database/{table.name}',
            title=f'{table.name} Table',
            content=content
        )

# Schedule regular sync
schedule.every(6).hours.do(sync_database_docs)
```

### Multi-Platform Publishing

Publish to multiple platforms simultaneously:

```go
// Publish to multiple destinations
func publishEverywhere(content *docai.Page) error {
    // Publish to Documentation.AI
    _, err := client.Pages.Create(ctx, content)
    if err != nil {
        return err
    }
    
    // Convert and publish to Confluence
    confluenceContent := convertToConfluence(content)
    err = confluenceClient.CreatePage(confluenceContent)
    if err != nil {
        return err
    }
    
    // Generate and commit to Git
    markdownContent := convertToMarkdown(content)
    return gitClient.CommitFile(content.Path+".md", markdownContent)
}
```
